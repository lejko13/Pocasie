# Pocasie (Mobile App)

Moderná mobilná aplikácia na zobrazenie počasia, vytvorená v JavaScript-e s využitím API od OpenWeather.

Aplikácia poskytuje:

- aktuálne počasie a predpoveď na 4 dni  
- prepínanie jednotiek (°C / °F)  
- responzívny moderný „glass UI“ dizajn optimalizovaný pre mobilné zariadenia  
- dynamické UI prispôsobené času (deň/noc)  
- prehľadné a intuitívne mobilné rozhranie  

Projekt je zameraný na čistý dizajn, používateľský komfort a prácu s externým API.

# Weather App (Mobile)

A modern mobile application for displaying weather information, built in JavaScript using the OpenWeather API.

The application provides:

- current weather and a 4-day forecast  
- unit switching (°C / °F)  
- responsive modern glass UI design optimized for mobile devices  
- dynamic UI adapting to time (day/night)  
- clean and intuitive mobile user interface  

The project focuses on clean design, user experience, and working with external APIs.
